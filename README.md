# 🗺️ 旅游行程计划统计表 · Travel Planner

一个零依赖、纯单文件的在线旅游行程规划工具。打开即用，数据自动保存在浏览器本地。

👉 **在线访问**：[https://codm-design.github.io/travel-planner/travel-planner.html](https://codm-design.github.io/travel-planner/travel-planner.html)

---

## ✨ 核心功能

### 📋 行程规划
- **分日管理**：按天组织行程，每天可添加任意条目
- **7 大分类**：交通 / 住宿 / 餐饮 / 门票 / 购物 / 娱乐 / 其他，颜色区分
- **就地编辑**：点单元格直接改，失焦自动保存
- **项目/地点 + 详情备注**：双行单元格，长文本也能完整显示
- **每天自动总览条**：列表形式展示当天的交通 + 住宿，一眼看清"怎么走 + 住哪"

### 💰 预算统计
- **5 张统计卡**：总预算 / 人均花费 / 日均花费 / 完成度 / 最大支出分类
- **支出分类柱状图**（横向，按金额从大到小排序）
- **每日花费趋势图**
- **人均分摊表**：自动按类别计算

### 🏨 预订信息
- **酒店预订板块**：名称 / 城市 / 入住退房（自动算晚数）/ 房型 / 确认号 / 总价 / 电话
- **机票预订板块**：航班号 / 航司 / 起降时间 / PNR / 座位号 / 票价
- 与行程主表独立，避免拥挤

### 🎨 实用工具
- **自定义颜色标记**：8 种预设色 + 原生取色器，标记重要/待确认/已预订等
- **行级独立锁定**🔒：已确认的条目单独锁上，其他照常编辑
- **完成状态切换**：✓ 已完成（红）/ ○ 未完成（绿），点一下切换
- **自定义排序**：手动 / 时间升降序 / 金额高低 / 按类型分组 / 未完成优先
- **🎒 出行清单**：默认 14 项（可扩展），勾选跟踪

### 📤 导入导出
- **🇳🇿 一键加载新西兰示例**（10 天完整行程 + 7 条酒店 + 4 段航班）
- **🇯🇵 日本关西 5 日游示例**
- **📊 Excel 导出**（5 个 sheet：行程明细 / 分类汇总 / 基本信息 / 酒店预订 / 机票预订）
- **💾 JSON 导出**（完整数据备份，跨设备迁移）

### 🔒 数据安全
- 所有数据存在浏览器 `localStorage`，**不上传任何服务器**
- 清理缓存 / 换浏览器前请先导出 JSON 备份

## 🛠 技术栈
- 单文件 HTML，零构建
- [Chart.js](https://www.chartjs.org/) 可视化
- [SheetJS](https://sheetjs.com/) (xlsx) 导出
- 纯原生 JavaScript，无任何框架
